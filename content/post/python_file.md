## ファイルの書き込み

モードに`'w'`を指定する。

```
with open('sample.txt', 'w') as f:
    f.write('Hello, world!\n')
```

## ファイルの読み込み

モードに`'r'`を指定する。

```
with open('sample.txt', 'r') as f:
    data = f.read()

print(data)
# 'Hello, world!'
```

## ファイルのデフォルト読み込み

モードは指定しなければ'r'となる。
そのため、読み込み時はモードの指定は不要。

```
with open('sample.txt') as f:
    data = f.read()

print(data)
# 'Hello, world!'
```

## ファイルの追加書き込み

モードに'a'を指定する。

```
with open('sample.txt', 'a') as f:
    f.write('Hello, again!')
```

追加書き込みがされていることが確認できる。

```
with open('sample.txt') as f:
    data = f.read()

print(data)
# Hello, world!
# Hello, again!
```
