## 英数字8文字のランダムパスワードを生成

```
>>> import random, string
>>> ''.join([random.choice(string.ascii_letters + string.digits) for i in range(8)])
'8GkJXlpr'
```

数字だけの場合は`string.digits`のみにし、長さを変更したい場合は`8`のところを変更すればよい。
