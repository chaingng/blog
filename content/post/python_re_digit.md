## 数字が１つだけの場合

`re.search()`で一致した文字列があればマッチオブジェクトを返す。

マッチオブジェクトから、`group()`でマッチした文字列を取得できる。

```
import re
re.search(r'[0-9]+', '4時間').group()
## '4'
```


## 数字が複数の場合

`re.findall()`で一致した文字列をすべてlistで返す。

```
import re
re.findall(r'[0-9]+', '4時間 50分')
# ['4', '50']
```
